3. สถาปัตยกรรมทางเทคนิค (Technical Architecture)
--------------------------------------------------------------------------------
- Frontend: React 18+, Vite, Tailwind CSS, Lucide Icons
- Backend: Express.js (Node.js) ทำงานในรูปแบบ Full-stack
- Data Persistence: 
    * Server-side: In-memory store พร้อมระบบสำรองข้อมูลลงไฟล์ db.json
    * Client-side: LocalStorage (Metadata) และ IndexedDB (Large Files/Images)
- Real-time Engine: WebSockets (ws) สำหรับการกระจายข้อมูลในทีมทันที
- Integrations:
    * AI Engine: Gemini API (Google GenAI)
    * Map Service: Google Maps Embed API
    * Cloud Storage: Google Sheets Integration (via Apps Script)
