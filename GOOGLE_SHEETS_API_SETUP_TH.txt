📘 คู่มือการตั้งค่า Google Sheets API (v4) - Direct Integration
ระบบ PEA PQ Smart Tracker
================================================================================

การเชื่อมต่อแบบ Direct API มีความเสถียรและปลอดภัยกว่าการใช้ Apps Script 
โดยใช้ Service Account ในการเข้าถึงข้อมูลโดยตรง

---

### 🛠 ขั้นตอนที่ 1: สร้าง Google Cloud Project และเปิดใช้งาน API
1. เข้าไปที่ Google Cloud Console (https://console.cloud.google.com/)
2. สร้างโปรเจกต์ใหม่ (New Project) หรือเลือกโปรเจกต์ที่มีอยู่
3. ไปที่เมนู "APIs & Services" > "Library"
4. ค้นหา "Google Sheets API" แล้วกดปุ่ม "Enable"

---

### 🔑 ขั้นตอนที่ 2: สร้าง Service Account และคีย์
1. ไปที่เมนู "APIs & Services" > "Credentials"
2. กดปุ่ม "Create Credentials" > เลือก "Service Account"
3. ตั้งชื่อ Service Account (เช่น pea-pq-tracker) แล้วกด "Create and Continue"
4. ในส่วน Role ให้เลือก "Editor" (หรือข้ามไปก่อนก็ได้) แล้วกด "Done"
5. ในตาราง Service Accounts ให้คลิกที่อีเมลที่เพิ่งสร้าง
6. ไปที่แถบ "Keys" > กด "Add Key" > "Create new key" > เลือก "JSON"
7. ไฟล์ JSON จะถูกดาวน์โหลดลงเครื่อง ให้เปิดไฟล์นี้ด้วย Text Editor

---

### 📄 ขั้นตอนที่ 3: เตรียม Google Sheet และให้สิทธิ์เข้าถึง
1. เปิด Google Sheet ที่ต้องการใช้งาน
2. คัดลอก "อีเมลของ Service Account" (จากไฟล์ JSON หรือหน้า Console)
3. กดปุ่ม "Share" ใน Google Sheet
4. วางอีเมลของ Service Account ลงไป และให้สิทธิ์เป็น "Editor"
5. คัดลอก "Spreadsheet ID" จาก URL ของ Sheet
   (เช่น https://docs.google.com/spreadsheets/d/[SPREADSHEET_ID]/edit)

---

### ⚙️ ขั้นตอนที่ 4: ตั้งค่า Environment Variables ใน AI Studio
นำข้อมูลจากไฟล์ JSON และ Google Sheet มาใส่ในเมนู Settings ของ AI Studio ดังนี้:

1. GOOGLE_SERVICE_ACCOUNT_EMAIL = (ค่า client_email ในไฟล์ JSON)
2. GOOGLE_PRIVATE_KEY = (ค่า private_key ในไฟล์ JSON)
   *หมายเหตุ: ต้องคัดลอกมาทั้งหมดรวมถึงส่วน -----BEGIN PRIVATE KEY-----*
3. GOOGLE_SHEET_ID = (Spreadsheet ID ที่คัดลอกมา)

---

### ✅ ขั้นตอนที่ 5: ทดสอบการเชื่อมต่อ
1. ไปที่เมนู "เครื่องมือ (Tools)" ในแอป
2. กดปุ่ม "Test Sync"
3. ระบบจะแจ้งว่า "เชื่อมต่อ Google Sheets API (v4) สำเร็จ!"
